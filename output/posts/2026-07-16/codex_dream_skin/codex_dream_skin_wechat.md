# Codex 也能换皮肤、养宠物？我找到了免费开源方案

最近在 AI 编程圈子里，我看到一个挺有意思的现象：有人在卖「Codex 皮肤/宠物」课程，价格从几十到几百不等。我点开看了下课程内容，核心其实就是两个开源仓库的安装和使用。换句话说，你愿意自己动手，这笔钱完全可以省下来。

这篇文章我就把这个免费路径完整复现一遍。你照着做，20 分钟内就能让 Codex 穿上新皮肤。

## 01 Codex 换肤，到底在换什么

Codex 是 OpenAI 推出的 AI 编程客户端，默认界面是一套干净但有点无聊的浅色风格。所谓「换肤」，不是修改它的模型能力，而是给这个客户端的界面换上自定义的视觉主题：背景图、配色方案、侧边栏样式、输入框装饰，有些主题还会加入可爱的动物或动漫形象，被大家戏称为「宠物」。

它解决的不是生产力问题，是「每天盯着同一个窗口几小时，越看越累」的问题。对长期坐在电脑前的人来说，界面顺眼一点，心情真的会好一点。

## 02 为什么有人能拿这个卖课

这件事能卖课，主要是因为信息差。

Codex 本身没有官方皮肤市场，换肤需要借助外部工具。普通用户第一次听到「CDP 注入」「运行 PowerShell 脚本」这些词会本能觉得复杂，而卖课的人只是把 GitHub 仓库的 README 翻译了一遍、录了个安装视频，然后打包出售。

但真正动手之后你会发现，整个过程就是下载、运行脚本、点切换。不需要写代码，也不需要懂前端。两个开源项目都把难度降到了最低。

## 03 两个方案，覆盖两种需求

我找到的两个项目定位不同，但都是免费开源。

**Fei-Away/Codex-Dream-Skin：开箱即用的主题包**

这个仓库更像一个「皮肤商店」的开源替代品。它内置了十几个主题，包括：

- 月薪喵：一只趴在电脑前打工的猫，适合自嘲打工人
- 中国水墨：山水、宣纸、毛笔质感
- 财神打工版：过年氛围拉满
- 初音未来、海贼王、红白科幻、舞台黑金等 IP/风格主题

Mac 和 Windows 都有对应脚本，下载后按平台运行即可。

**CodeDrobe/skills：更系统的换肤框架**

这个项目走得更远。它提供了一套完整的 Codex 主题 Skill，包含：

- 跨平台的 `@codedrobe/core` 运行时
- 可安装的 `codedrobe-theme` Skill
- AI 辅助生成主题的工作流
- 支持 Codex 和 Tencent WorkBuddy 两个应用

如果你只是换个皮肤，Dream Skin 更轻量；如果你想自己设计主题、甚至批量生成不同风格，CodeDrobe 更适合。

## 04 我的安装复现过程

我先用 Windows 把 Dream Skin 跑了一遍，全程不到 10 分钟。

**第一步，下载仓库。** 可以直接 `git clone`，也可以下载 zip 解压。仓库按平台分了 `macos/` 和 `windows/` 两个目录。

**第二步，运行安装脚本。** Windows 下以管理员身份打开 PowerShell，进 `windows/` 目录，先执行 `scripts/install-dream-skin.ps1`，再执行 `start-dream-skin.ps1`。Mac 更简单，双击 `Install Codex Dream Skin.command` 就行。

**第三步，打开 Codex 切换主题。** 脚本会启动一个皮肤管理器界面，左侧是皮肤库，中间显示已安装的皮肤。挑一个主题点「一键切换」，界面立刻生效。点「恢复原厂」就能回到默认外观。

如果想换另一个主题，不用重新安装，直接再点「一键切换」即可。

CodeDrobe 的路线稍微不同：先全局安装核心运行时 `npm install --global @codedrobe/core`，然后用 `npx skills add CodeDrobe/skills --skill codedrobe-theme --global --agent codex` 安装主题 Skill，最后用 `codedrobe apply --app codex --theme /path/to/theme.codedrobe-theme` 应用主题。它还多了一个 `codedrobe verify` 命令，可以截图验证主题有没有正确应用。

## 05 安全边界：能碰和不能碰的

虽然这两个项目都不改官方安装包，但还是要把安全边界说清楚：

- **只拉可信仓库。** 主题会注入到客户端界面里，来路不明的主题包可能夹带恶意代码。
- **只在本地跑。** CDP 只绑定 127.0.0.1，主题运行期间不要同时运行来路不明的本机程序。
- **不要把换肤当成性能优化。** 它只是外观美化，不会让 Codex 写代码更快。
- **商用再分发需谨慎。** 仓库效果图里的动漫、IP 形象只是主题示意，公开商用或二次分发要自行确认肖像权和商标授权。

## 06 我的选型建议

如果你只是想让自己每天用的 Codex 好看一点，选 **Dream Skin**。它主题多、安装简单、一键切换，10 分钟搞定。

如果你想深入玩主题、甚至自己做一套主题卖或分享，选 **CodeDrobe**。它的生态更完整，有 AI 生成主题的能力，未来可能会形成一个真正的主题市场。

对我来说，目前 Dream Skin 已经够用。我换了一个「月薪喵」主题，看着屏幕上那只同样在电脑前打工的猫，写代码的心情莫名好了点。

## 07 写在最后

回到开头那个卖课的现象。我的态度一直很简单：工具本身免费，课程就不应该成为入门门槛。尤其是这种「跟着 README 做一遍就会」的东西，与其花钱看别人演示，不如自己花 20 分钟动手。

Codex 换肤这件事，免费开源方案已经做得很好。你完全可以自己试试，然后再决定要不要深入。
