# 别手动刷热点了，让AI盯大佬推特

做内容最烦的就是每天想「今天写啥」。手动刷热点又慢，还容易被算法投喂二手货——等你刷到，已经全网都在写了。

我搞了个 0 成本的法子：用一套免费工具（opencli，走你自己浏览器里已有的登录态，一分钱不花），把几个我常跟的 AI 大佬的 X / Reddit 实时抓下来，配几条规则过滤，每天自动给我递一批候选选题。

关键不是「抓取」多难，是「免费还不封号」——它复用你本机浏览器的登录态，不是跟平台硬刚的爬虫，所以不拦。

我实际这么用：
1. 列 10 个值得跟的人（做 AI 工具的、爱分享 workflow 的）
2. 用 opencli 拉他们的 timeline / 某个 subreddit 的新帖
3. 关键词过滤，只要提到 agent / workflow / 开源 的
4. 每天早上扫一眼，挑能落地的写

效果：选题库从「刷到啥算啥」变成「我主动盯一手信源」，而且 0 元。

提醒一句：免费 ≠ 随便造。节奏慢一点、别硬刚限流，细水长流才稳。
