# 22万星Agent神库我留下了

收藏夹里 GitHub 库吃灰的不少。22 万星 harness，我读完就留下了。

它叫 affaan-m/ECC，是一整套优化系统。把几项能力打包好。而且跨好几个主流 harness 通用。

我重点看了三块。

其一，真能省 token。文档里两块讲省 token 和存记忆。讲用 hooks 自动存读上下文。长会话不用每次从头重灌。这思路跟我半自动攒素材库一致。会话不丢前情，挺戳我。

其二，安全有兜底。AgentShield 做沙箱加净化。Agent 跑命令不裸奔。我之前让 Agent 裸跑踩过雷。所以这块对我很有用。

其三，它不硬塞全家桶。261 个 skills，别全装。我目前只拿 memory 和 skills 两块。够用就行。

我的不同意见：新手别一上来全接。开源 MIT 免费，够用。Pro 那个付费版（托管私有库）我没买，用不上。接上 memory，token 就明显省了。

顺带说一句，这库我只扒了 README，没全套实测。上面理解按文档来，不是打包票。
