## 三、怎么用
- 风向标（一）喂 `ziliaoku-signal`：什么工具正在火 → 做「XX 正在爆火」选题。
- 正文库（二）喂 `ziliaoku-extract`：高 star 库 README → 做「神库推荐 / 工具盘点」选题（自带 star 背书）。

> 红线：标题可借 star 背书（「为什么 14 万人在用」），但不写收益数字、不绝对化。
