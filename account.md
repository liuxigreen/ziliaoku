# 账号档案

## 定位
用 AI Agent 做内容工作流的实操博主

## 人设三关键词
实操、接地气、不画饼

## 变现路径
小红书引流 → 私域/知识星球 → 选题库订阅（99-299元/年）

## 内容红线
- 不提具体收益数字
- 不发绝对化用语
- 不硬引流

## 写作调性
- 说人话，不端着
- 只分享自己跑过的东西
- 踩过的坑大方讲，不包装成成功经验

## 定位句模板（一句话说清"我是谁/给谁/解决什么"）
我是【用 AI Agent 做内容工作流的实操博主】，专门帮【想用 AI 提效但不想被工具淹没的创作者】，解决【内容生产慢、素材散、不会搭工作流】的问题。
> 用法：对外介绍 / 私域欢迎语 / 简介直接套。避免"全领域 AI 博主"式空泛。

## 门面系统（账号第一眼的信任基建）
- **昵称**：带"实操 / 工作流"暗示，别叫"XXAI"（太泛、无辨识）
- **简介**：定位句压缩版 + "只分享跑过的" + 更新频率（如"每周 3 更"）
- **开场白 / 置顶**：一句话人设 + "你在这能看到什么"（新粉第一条可见）
- **内容支柱（3 根，轮换不偏）**：① 工具 / Git 拆解（引流）② 爆款写法拆解（引流+信任）③ 自身复盘 / 观点（信任+转化）
- **关键词宇宙**：AI Agent / 内容工作流 / 自动化写作 / 半自动发帖 / 选题库（标题与简介高频复用，强化搜索命中）
