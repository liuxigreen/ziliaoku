## 周节奏（入驻后常态）

- 小红书 3-4 篇/周，公众号 1-2 篇/周；时段优先 12:30 / 19:30 / 21:00。
- 同平台间隔 ≥ 48h；转化型不连续（间隔 ≥ 2 篇其他类型）。
- 养号期（前 14 天）发布 ≤ 3 篇/周，不挂任何外链、不提私域。
- 排期只是建议，最终发布由用户终审拍板（自动化的是排期，不是决策权）。
