# Cursor 完整教程 2026：AI 代码编辑器从安装到精通（图解）
...
Cursor 怎么用？这篇完整教程带你从安装配置到 Tab 补全、Cmd+K 行内编辑、Chat、Agent 跨文件开发、Project Rules，配图解和实战流程，新手也能上手这款 AI 代码编辑器。
...
Cursor 是一个 AI 代码编辑器——你可以把它理解成"装了超强 AI 的 VS Code"。它基于 VS Code 打造，所以你的插件、快捷键、主题几乎都能直接用，但在此之上深度集成了 AI：能补全整段代码、按你的话改代码、读懂你整个项目，甚至自己跨多个文件完成一个功能。
...
这篇我（dfkai）带你从安装到把每个核心功能用顺，再到用 Agent 模式真正干活。不管你是程序员还是刚学编程的新手，跟着走一遍就能上手。
...
## 第 1 步：安装与配置
...
1. 去 Cursor 官网（cursor.com）下载对应系统的版本并安装。
2. 首次打开会问要不要从 VS Code 导入配置——选是，你的插件、快捷键、主题一键搬过来。
3. 注册登录（有免费额度，Pro 版额度更多）。
4. 在设置里能选用哪个模型（Claude、GPT 等），写代码场景 Claude 系列通常体验很好。
...
## 第 2 步：搞懂几个核心 AI 功能（最关键）
...
Cursor 的 AI 不是一个按钮，而是几种各司其职的能力。先看一张"该用哪个"的速查图：
...
- Tab 自动补全：你边写，它边预测接下来的代码（常常是整段、跨行），按 Tab 接受。这是用得最频繁、最省手的功能。
- Cmd+K（Windows 是 Ctrl+K）行内编辑：选中一段代码或把光标停在某处，按 Cmd+K，用大白话说"把这个函数改成异步""加上错误处理"，它就地改给你看，你确认 diff 再接受。
- Chat 对话（Cmd+L）：右侧聊天窗，问关于你这个项目的问题——"这个 bug 可能出在哪""这段逻辑是干嘛的"。它能读你的代码库来回答，不是泛泛而谈。
- Agent 模式：最强的一档。你给一个目标（"给登录页加上记住密码功能"），它自己规划 + 跨多个文件改 + 跑命令，最后把所有改动以 diff 呈现给你审。
- @ 引用：在输入框打 @，可以精确引用某个文件、某段代码、某个文档甚至网页，让 AI 聚焦在你指的范围，回答更准。
...
## 第 3 步：用 Agent 模式真正干一件事
...
新手最容易被惊艳的就是 Agent。典型流程：
...
1. 描述需求：用自然语言把要做的事说清楚，越具体越好（"在首页加一个深色模式切换按钮，状态存到 localStorage"）。
2. @ 引用相关文件：把相关的组件、配置 @ 进来，给它上下文。
3. 它规划并改：Agent 会列出计划、动手改多个文件、必要时运行命令。
4. 逐项审 diff：每个改动你都能看到 before / after，这一步别偷懒——AI 会犯错，你是把关人。
5. 接受或纠正：满意就接受；不对就追加一句"按钮要放右上角"让它改。
6. 跑起来验证：最后一定亲自运行测试，别盲信。
...
## 第 4 步：用 Project Rules 让 AI 更懂你的项目
...
想让 AI 始终遵守你项目的规范（用什么框架、什么代码风格、不准用什么），可以在项目根目录建一个规则文件（新版叫 Project Rules，老版是 .cursorrules）。例如：
...
```
- 这是一个 Next.js + TypeScript 项目，使用 App Router
- 组件一律用函数式 + 具名导出，不要用 default export
- 样式只用 Tailwind，不要写行内 style
- 不要引入新依赖，除非我明确同意
- 注释和回答都用中文

```
...
写好后，Cursor 每次生成代码都会参考这些规则，省得你反复纠正。
...
## 提效技巧
...
- 先 @ 再问：给足上下文，回答质量天差地别。
- 小步快跑：让 Agent 一次只做一件事，比一句话让它改一大堆更可控、更好审。
- 善用 Cmd+K 写测试 / 重构：选中代码说"给这个函数写单元测试"非常香。
- 不确定就先用 Chat 问，想清楚再让 Agent 动手。
...
## 常见坑
...
- 盲目接受 AI 的改动：最大的坑。一定逐项看 diff、跑测试。
- 上下文给太少：不 @ 文件就让它改，它只能猜，容易跑偏。
- 一句话让它改太多：改动面太大，难审也难回滚，请拆小。
- 忘了它会过时：AI 可能用了旧 API，关键处自己核对官方文档。
...
Cursor 的精髓是：Tab 补全省手、Cmd+K 局部改、Chat 问代码库、Agent 跨文件干活，再用 Rules 约束风格。把这几个用顺，AI 就从"玩具"变成你每天离不开的生产力工具。记住一条铁律——AI 负责写，你负责审。
...
Q：不会编程能用 Cursor 吗？
...
能。用 Agent 模式描述需求，它能帮你从零搭小项目；但能看懂、会审改动会让你走得更远。
...
Cursor 基于 VS Code 构建，插件、快捷键、主题基本通用，相当于"AI 增强版 VS Code"。
...
Cursor 和 Claude Code
...
Cursor 是图形界面的 AI 编辑器，适合可视化地写改代码；Claude Code 是命令行 Agent，更适合自动化和终端工作流。两者可以配合用。
