一个普通上班族的真实副业经历，没有背景、没有资源，全靠方法和执行力
...
我叫李明，坐标成都，主业是一家小公司的行政，月薪5000块，扣完社保到手4300。三年没涨过工资，领导画的饼比我脸还大。
...
2026年1月，我开始接触AI副业。到3月底，三个月时间，副业总收入81200元。最高的一个月拿了34000，最低的一个月也有22000。
...
今天把方法全部公开，包括我踩过的坑、用过的工具、走过的弯路。全文很长，建议先收藏，慢慢看。
...
那段时间我疯狂搜索"副业""兼职""普通人怎么赚钱"，看了几百篇文章，90%都是割韭菜的——要么让你交学费学什么"无货源电商"，要么让你买什么"赚钱系统"。
...
我突然意识到：这就是信息差。大部分人还不知道AI能干什么，而我已经可以靠"帮别人用AI"来赚钱了。
...
# 【第二章：我试过的5条路，3条是坑】
...
# 坑1：AI绘画卖图
...
听起来很美对吧？用Midjourney生成图片，然后卖给需要素材的人。我花了两周时间研究，生成了上千张图，挂到各个素材平台上。
...
结果：三个月总共卖出去7张，赚了35块钱。
...
为什么不行？因为AI绘画已经卷到极致了，免费素材铺天盖地，没人愿意为AI生成的图片付费。而且版权问题一直悬而未决，很多平台直接禁止AI图片上架。
...
# 坑2：AI写小说
...
用ChatGPT写网络小说，发到番茄、七猫等平台赚稿费。我试了一个月，写了8万字。
...
结果：阅读量惨不忍睹，平台判定为"低质内容"，直接限流。
...
为什么不行？AI写的小说有一个致命问题——没有"人味儿"。读者一眼就能看出来是AI写的，因为遣词造句太"正确"了，没有口语化的表达，没有情绪的起伏。平台算法也在升级，AI生成内容的识别越来越精准。
...
# 坑3：AI配音
...
用AI语音工具生成配音，接短视频旁白的单子。我在某宝开了店，定价极低想走量。
...
结果：卷不过专业工作室，他们有更贵的工具、更丰富的音色，价格还比我低。一个月接了3单，赚了150块，还不够电费的。
...
这三条路让我明白一个道理：AI副业的核心不是"用AI"，而是"用AI解决别人解决不了的问题"。
...
# 【第三章：真正赚钱的3个方向】
...
踩完坑之后，我换了思路。不再想"AI能做什么"，而是想"别人需要什么，AI能不能帮我做到"。
...

我的做法：用AI工具（主要是豆包和通义千问）生成初稿，然后自己润色修改，加上行业术语和真实案例，让文案读起来像是"懂行的人写的"。
...
定价策略：单篇100-300元，比专业公司便宜70%，比AI直出的质量高几个档次。
...
获客渠道：我在闲鱼、小红书、本地商家群发了免费试写的帖子，第一批客户是靠"免费写一篇，满意再付费"拉来的。口碑起来之后，客户转介绍占了60%以上。
...
关键技巧：不要把AI生成的内容直接交付。AI是"原材料供应商"，你是"厨师"。原材料再好，不做饭也吃不了。我会花30%的时间用AI生成，70%的时间做人工优化——调整语气、补充案例、加入情感共鸣点。
...
方向二：AI辅助短视频脚本+剪辑（月入10000-20000）
...
一个做餐饮的朋友问我能不能帮他拍抖音，我说我不会拍视频，但可以帮他写脚本。结果我用AI生成了一个探店脚本，他拍出来之后播放量破了10万，直接带火了他的店。
...
1. 用AI分析同行业爆款视频的结构和话术

2. 用AI生成脚本框架，我再加入本地化元素和真实体验

3. 用剪映的AI功能辅助剪辑（自动字幕、智能配乐、一键成片）

4. 交付成品视频，每条收费300-800元
...
这个方向的收入上限很高，因为老板们一旦看到效果，就会持续下单。我现在有6个稳定客户，每月固定产出30-40条视频。
...
方向三：AI培训+咨询（月入15000-34000）
...
我先是免费分享经验，后来发现需求太大了，干脆开了一个线上的"AI副业实战课"。课程内容就是我自己的真实经历——用什么工具、怎么获客、怎么定价、怎么避坑。
...
定价199元/人，第一期招了40个人，收入将近8000元。口碑发酵后，第二期120人，第三期200人。到第三个月，光培训收入就有34000元。
...
关键心得：教别人赚钱，是最好赚的钱。但前提是你自己真的赚到了，有真实案例，而不是纸上谈兵。我每一节课都展示自己的真实收入截图、客户聊天记录、作品案例，这种"真实感"是最好的营销。
...
# 【第四章：给想做AI副业的朋友5条建议】
...
第一，不要辞职。副业是"锦上添花"，不是"破釜沉舟"。先用业余时间跑通模式，等副业收入稳定超过主业3倍以上，再考虑全职。
...
第二，不要买课。市面上90%的"AI赚钱课"都是割韭菜。真正有用的信息，B站、小红书、知乎上全有，而且免费。你需要的不是课程，是行动力。
...
第三，从最小的切口进去。不要一上来就想做"AI创业"，先从帮一个朋友写一篇文案、帮一个老板剪一条视频开始。小钱赚到了，大钱自然来。
...
第四，重视客户关系。AI副业的复购率极高，一个满意的老客户能带来5-10个新客户。服务好每一个人，比开发新客户重要10倍。
...
第五，持续学习。AI工具更新迭代极快，三个月前的方法可能已经过时了。每天花30分钟关注行业动态，是最低成本的自我投资。
...
回头看这三个月，最大的感受是：普通人赚钱，靠的不是天赋，是信息差和执行力。
...
当大部分人还在观望的时候，你已经开始做了；当大部分人开始做的时候，你已经跑通了。这就是先发优势。
...
AI时代最大的红利，不是技术本身，而是"大部分人还没反应过来"这个时间窗口。
