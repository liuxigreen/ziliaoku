# 小红书新手冷启动：WorkBuddy 完整操作手册 + 所有模板

作者：文子🔆 @Eejoylove
工具：WorkBuddy（腾讯 AI Agent 桌面工作台）
适用人群：0基础新手，想低粉高变现
日期：2026年7月
使用说明：所有 Prompt 均可直接复制到 WorkBuddy 使用，只需替换{}里的关键词。

## 一、先想清楚怎么变现（最重要一步）

思考框架：
- 你的产品/服务是什么？
- 变现路径：接商单 / 开店卖货 / 挂链接带货 / 导流私域成交
- 粉丝量决定定价
- 内容方向由变现方式倒推（美食探店 vs 知识科普）
- 选择自己喜欢且能坚持的赛道

记住：先定赚钱方式，再做账号。别内容做起来才发现赛道难变现。

## 二、找低粉爆款，直接拆（核心学习法）

找爆款四步：
1. 搜索核心关键词（例：AI工具、控糖食谱、三伏天养生）
2. 切换「最新」排序
3. 挑选刚发布、点赞/浏览量明显高于同期的笔记（低粉几百粉、高赞高曝光）
4. 拆解标题、开头、结构、话题标签

WorkBuddy Prompt 模板（拆爆款）：
我给你发了5-10篇小红书同赛道低粉爆款笔记。请按以下要求拆解：
1. 总结标题公式（列出7种常见类型）
2. 拆解开头3句结构
3. 提取整体内容骨架（段落逻辑）
4. 提取常用话题标签和结尾方式
5. 输出一个可复用的「内容公式」
只输出结构，不要抄原文句子。

操作提示：把爆款标题+内容截图或复制发给它，截图重点框「总结出的内容公式」部分。
把公式保存进 WorkBuddy 记忆文件（.workbuddy/memory/）。

## 三、给 WorkBuddy 建「账号档案」

Prompt 模板（建档案）：
请记住以下账号信息，并写入记忆文件（.workbuddy/memory/账号档案.md）：
账号定位：{例如：文科生用AI工具提升效率}
变现模式：{导流私域成交}
人设调性：{专业、亲切、真实，3个关键词}
内容边界：不提具体收益数字、不直接引流微信、不发违规内容
已有数据：{当前粉丝数、笔记数}
以后所有生成的内容都要严格符合以上定位和风格。

示例（作者风格）： 账号定位：文科生如何用AI高效创作
人设：真实分享、干货落地、像朋友聊天

## 四、建选题库（解决每天想选题）

7种标题公式 Prompt：
请为小红书{你的赛道}内容生成7种标题公式模板，每种公式提供5个示例：
1. 痛点型
2. 数字型
3. 对比型
4. 稀缺型
5. 共鸣型
6. 资源型
7. 反常识型
每个公式都要带占位符{}，方便后续填空。

生成选题 Prompt：
使用上面建立的标题公式库，为{你的赛道}生成35个选题（每种类型5个）。
要求：真实、自然、无营销味、可直接使用。

## 五、选题 + 生成内容

Prompt 示例：
从选题库中推荐3-5个适合新号第一周发布的选题。
然后用我们账号档案的风格，为选题「{具体选题}」生成完整笔记：
- 标题
- 开头3句（带钩子）
- 正文结构（按拆解的爆款骨架）
- 结尾 + 话题标签
- 去AI味，加入个人真实经历口吻

养号提醒：新号先刷3-5天同赛道内容，建立活人轨迹，再发笔记。

## 六、去 AI 味 + 合规检查（防限流）

去AI味 Prompt：
用以下范本的风格和语气，改写我给你的这篇初稿：
{粘贴你写得最好的笔记作为范本}
要求：
1. 加入真实个人经历和口语化表达
2. 保持温暖、有温度，像朋友聊天
3. 去除工整书面感

合规检查 Prompt：
检查这篇笔记是否包含违规词（绝对化、最/第一/唯一、收益承诺、直接引流等）。
列出问题并给出修改建议。

## 七、排期 + 复盘（长期运营）

30天排期 Prompt：
根据我们的账号档案，制定30天小红书发文排期表，分3阶段：
1. 养号期
2. 标签识别期
3. 稳定变现期
每一天标明：发布时间建议、内容类型、选题方向、标题公式。

每周复盘 Prompt：
我给你以下数据：
小眼睛/点击率/点赞/收藏/评论/涨粉
请填入复盘看板模板，分析规律、给出下周优化建议，并记录进记忆文件。

## 额外必做事项
1. 养号：注册后3-5天只浏览、点赞、收藏、少量评论同赛道内容。
2. 两个手机：研究手机（看内容、拆爆款） + 发文手机（推荐二手iOS，流量更稳）。
3. 账号名 + 简介：参考爆款，简洁体现定位和价值。

## 最后提醒 & 为什么选 WorkBuddy
- 工具是杠杆，核心是你自己的方法论和真实经历。
- 记忆文件是最大资产（公式、踩坑、数据全积累）。
- AI 生成必须人工过一遍，尤其是标题和前3句。
- 先跑通一套自己的打法，再换工具也通用。

完整流程总结（7步）：
1. 先定变现方式
2. 找低粉爆款拆公式
3. 建账号档案
4. 建选题库
5. 选题生成内容
6. 去AI味 + 合规
7. 排期 + 每周复盘

行动建议：边做边优化，行动打败一切。做起来后可教别人，创造更多价值。
