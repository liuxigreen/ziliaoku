# AiToEarn — 一站式 AI 内容营销智能体

> Monetize · Publish · Engage · Create。帮助 OPC（一人公司）、创作者、品牌与企业在全球主流平台构建、分发并变现内容。

支持渠道：抖音、小红书(Rednote)、快手、B站、视频号、TikTok、YouTube、Facebook、Instagram、Threads、Twitter(X)、Pinterest、LinkedIn。

## 四大 Agent
- **Create（创作）**：调视频/图片模型（Grok、Veo、Seedance、Nano Banana 等）批量生成图文视频。
- **Publish（发布）**：一键分发 14 个平台 + 日历排期。
- **Engage（互动）**：自动点赞/收藏/关注 + **AI 智能回复评论** + 评论挖掘（识别「求链接」「怎么买」高转化信号）+ 品牌监测（浏览器插件）。
- **Monetize（变现）**：自带 CPS/CPE/CPM 内容交易市场，创作者接商家推广任务赚钱。

## 部署方式（5 种）
网页直接用 / 接 OpenClaw（MCP 插件）/ 接 Claude·Cursor（MCP）/ Docker 一键部署 / 源码。需 API Key（中国版 aitoearn.cn / 国际版 aitoearn.ai）。

## 配置 Relay（发布需登录社媒账号 OAuth）
配 Relay 后借用官方 aitoearn.ai 凭据完成 OAuth 授权，不需自申请开发者账号。

## 风险点（分析视角）
- 全自动发帖 + 互动 = 平台高风控行为，账号资产风险。
- 「AI 回评论」伤真人感 / 人设。
- 变现市场非躺赚，需账号权重 + 内容质量 + 合规。

## 与账号定位的关系
直接对标「用 AI Agent 做内容工作流的实操博主」：它的「全自动批量铺量」恰好反衬本账号的「human-in-loop + 实操不画饼 + 账号安全」差异化。
