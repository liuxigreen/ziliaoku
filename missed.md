# 漏网追踪记录

> 记录用户人肉刷到、但系统应该发现却没发现的好内容。
> 连续两周在同一平台漏网 → 考虑启用第6入口。

## 记录格式

```
- **日期**: YYYY-MM-DD
- **标题**: 文章/帖子标题
- **来源平台**: 系统漏掉的平台（如：即刻、公众号榜单、某论坛）
- **为什么系统漏了**: 系统哪个环节没覆盖到（如：聚合站没收录、关键词没命中、赛道词不匹配）
- **链接**: URL（可选）
```

## 记录示例

```
- **日期**: 2026-07-08
- **标题**: Claude 4 新特性深度解析
- **来源平台**: 即刻
- **为什么系统漏了**: 即刻不在现有5个入口中，聚合站未覆盖
- **链接**: https://okjike.com/...
```

## 触发条件

当满足以下任一条件时，考虑启用第6入口：
- 同一平台连续 2 周漏网 ≥ 3 条
- 漏网内容在周复盘中被评为高价值（collect 级别）
